MiniWheg

The challenge was to design an insect-inspired mobility robot that assumes a base configuration of the MiniWheg.

Design development can be categorised into four main areas, as shown. Improvements were made through several iterations for increased mobility and durability in uneven outdoor terrains, with added camouflage in grassy areas. Decisions were guided by engineering analysis through calculations, computed using Microsoft Excel and through simulations, using the CAD software Solidworks.

engineering

Various calculations were carried throughout the project, including the application of:
- conservation of moments
- the Lewis formula (gear stress)
- Contitech synchronous belts
- shear force and bending moment diagrams
- torque calculations
- ackerman angles

Through an iterative process, finite element analysis via Solidworks simulations (static studies) was used to identify and improve on areas of stress concentrators. Motion studies were used to understand how parts would interact with each other, for example in the steering system.
