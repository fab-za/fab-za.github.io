rise Dock, Pad, Band

User-centred design of an alarm clock and alerting system for d/Deaf and hard of hearing. Collaboration with Jemima Bradley, Alberto Montemiglio and Adedunke Oyenuga.

Date: 2020
Team Role: Chief Creative Officer
Main: creative direction, compiling 50+ portfolio pages using Adobe suite, market analysis and breakdown of parent brand, fabrication of orthographic drawings, assembly/production cost analysis, creation of user guide, user interaction mapping
Sub: compliance, bill of materials and CMF specification

brief

Design and engineer a battery powered hand-tool for home, garden or educational use that not only appeals to the mass market, but also meets the needs of a specific (underserved) user group. Each team must deliver a compelling and commercially viable product design solution, based on genuine user insights, realised to fully working 3D modelled prototype, supported by a detailed production data package.

outcomes

The Rise family consists of three members, the dock, pad, and band, each of which can alert the user with more than just sound. The Rise band integrates with existing Garmin smart watches and contains a specialised NFC module, which the user can tap on a pre-programmed Rise sticker to automatically set a timer. When the timer is finished, the Garmin watch will display the corresponding sticker colour, and the custom set alerts will go off. The Rise Pad acts as a vibrating alarm, which can be placed beneath a pillow, whilst the Rise Dock produces light and sound alerts.
