This Site

Coded portfolio site from scratch and hosted on Github.

Date: 2020
Individual Project

brief

Knowing that I needed a portfolio to showcase my work during internship applications, I decided it was the perfect opportunity to further the HTML, CSS and Javascript skills I learned during the Weather With Me project.

outcomes

The result is a fully functional and responsive static website with an infinite scroll landing page showing my experience and projects as well as individual pages with further details. This was built entirely in pure code which was self-taught through a range of online resources and coded from a mobile first design approach. Some interesting features include navigation bar colour changes on scroll and outsourced dynamic site elements like the contact form.
